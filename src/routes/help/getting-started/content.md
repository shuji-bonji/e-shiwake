# はじめに

## e-shiwake とは

e-shiwake（電子仕訳）は、フリーランス・個人事業主向けの仕訳入力＆証憑管理アプリです。

以下の特徴があります。

- **ローカルファースト**：サーバー不要、データはブラウザ内に保存
- **電帳法対応**：日本の電子帳簿保存法の検索要件に準拠
- **証憑管理**：PDFファイルを仕訳に紐付けて管理
- **PWA対応**：オフラインでも動作、アプリとしてインストール可能

## 対象ユーザー

- 日本のフリーランス・個人事業主
- 確定申告を自分で行う方
- クラウド会計の月額課金を避けたい方
- シンプルな仕訳入力ツールを探している方

## 基本的な使い方

1. **年度を選択**
   サイドバーから対象の会計年度を選択します。

2. **仕訳を入力**
   「新規仕訳」ボタンから仕訳を追加します。

3. **証憑を紐付け**
   PDFファイルをドラッグ＆ドロップで仕訳に紐付けます（元ファイルはそのまま残ります）。

4. **データをエクスポート**
   必要に応じてJSON/CSV形式でエクスポートします。

## データの保存について

> **INFO**: e-shiwakeのデータはすべてブラウザ内（IndexedDB）に保存されます。サーバーへの通信は行いません。

> **WARNING**: ブラウザのデータを消去すると、仕訳データも削除されます。定期的なバックアップをお勧めします。

## 推奨環境

- Chrome / Edge（最新版）
- Safari（最新版）
- Firefox（最新版）

※ 一部の機能（証憑PDFファイル保存先の選択など）はChrome/Edgeでのみ利用可能です
