# 青色申告決算書

## 青色申告決算書とは

青色申告決算書は、個人事業主が確定申告で提出する書類で、
1年間の事業の収支を記載します。一般用は4ページで構成されています。

> **INFO**: 青色申告を行うと、最大65万円の特別控除が受けられます。e-shiwakeで作成したデータを元に、決算書を作成できます。

## 4ページの構成

| ページ | 内容 | 説明 |
|--------|------|------|
| 1ページ目 | 損益計算書 | 売上・経費・所得金額の計算 |
| 2ページ目 | 月別売上・経費 | 月別の売上・仕入金額と経費内訳 |
| 3ページ目 | 減価償却費の計算 | 固定資産の減価償却明細 |
| 4ページ目 | 貸借対照表 | 資産・負債・資本の状況 |

## 基本的な使い方

1. サイドバーから「青色申告決算書」を選択
2. タブで各ページを切り替えて内容を確認
3. 「設定」ボタンで事業者情報や控除額を設定
4. 「印刷」ボタンで印刷またはPDF保存

## 設定項目

「設定」ボタンから以下の項目を設定できます。

| 項目 | 説明 |
|------|------|
| 氏名 | 申告者の氏名 |
| 屋号 | 事業の屋号（任意） |
| 住所 | 事業所の住所 |
| 事業の種類 | 業種（例：システム開発業） |
| 青色申告特別控除 | 65万円/55万円/10万円から選択 |
| 期首/期末棚卸高 | 商品・製品がある場合 |
| 事業主貸/借 | 事業とプライベート間の資金移動 |

## 青色申告特別控除

控除額は申告方法によって異なります。

| 控除額 | 条件 |
|--------|------|
| 65万円 | e-Tax + 複式簿記 + 貸借対照表 |
| 55万円 | 紙提出 + 複式簿記 + 貸借対照表 |
| 10万円 | 簡易簿記 |

> **TIP**: e-shiwakeは複式簿記で記帳されるため、e-Taxで申告すれば65万円控除が適用できます。

## 1ページ目：損益計算書

仕訳データから自動で以下の金額が計算されます。

- 売上（収入）金額
- 売上原価（期首棚卸 + 仕入 - 期末棚卸）
- 経費の内訳と合計
- 青色申告特別控除前の所得金額
- 所得金額（青色申告特別控除後）

## 2ページ目：月別売上・経費

仕訳データから月別の売上・仕入金額が自動集計されます。

- 月別売上金額（1月〜12月）
- 月別仕入金額（1月〜12月）
- 雑収入、給料賃金、地代家賃の合計

## 3ページ目：減価償却費の計算

固定資産台帳に登録した資産の減価償却情報が表示されます。

> **INFO**: 固定資産台帳に資産を登録しておく必要があります。未登録の場合は「減価償却資産がありません」と表示されます。

## 4ページ目：貸借対照表

期末時点の資産・負債・資本の状況が表示されます。

- 流動資産（現金、預金、売掛金など）
- 固定資産（建物、車両、備品など）
- 流動負債（買掛金、未払金など）
- 固定負債（借入金など）
- 資本（元入金、事業主貸/借）

> **WARNING**: 貸借バランスが一致しない場合は警告が表示されます。仕訳の入力漏れや誤りがないか確認してください。

## 印刷・CSV出力

作成した決算書を印刷またはCSV出力できます。

- 「印刷」ボタン：全4ページを印刷（PDFとして保存可能）
- 「CSV出力」ボタン：サマリーデータをCSVでダウンロード

> **TIP**: 印刷時は全4ページが自動的に出力されます。ブラウザの印刷ダイアログで「PDFとして保存」を選択すると、PDFファイルになります。

## バリデーション

データに問題がある場合、画面上部に警告メッセージが表示されます。

- 月別売上合計と損益計算書の売上が一致しない場合
- 貸借対照表のバランスが一致しない場合
- 青色申告特別控除額が不正な場合
