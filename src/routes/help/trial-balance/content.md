# 試算表

## 試算表とは

試算表は、すべての勘定科目の借方・貸方の合計と残高を一覧表示する帳簿です。
決算前に仕訳の正確性を確認するために使用します。

> **INFO**: 複式簿記では、借方合計と貸方合計は必ず一致します。一致していない場合は仕訳に誤りがあります。

## 基本的な使い方

1. サイドバーから「試算表」を選択
2. 画面上部のドロップダウンで年度を選択
3. 勘定科目グループごとに集計された一覧が表示されます

## 表示モード

2つの表示モードを切り替えられます。

| モード         | 表示内容                               | 用途           |
| -------------- | -------------------------------------- | -------------- |
| 合計残高試算表 | 借方合計・貸方合計・借方残高・貸方残高 | 詳細な確認     |
| 残高試算表     | 借方残高・貸方残高のみ                 | シンプルな確認 |

## 勘定科目グループ

科目は以下の5つのグループに分類されて表示されます。

| グループ | 主な科目例                   | 通常の残高位置 |
| -------- | ---------------------------- | -------------- |
| 資産     | 現金、普通預金、売掛金       | 借方           |
| 負債     | 買掛金、未払金、借入金       | 貸方           |
| 純資産   | 元入金、事業主貸・借         | 貸方           |
| 収益     | 売上高、雑収入               | 貸方           |
| 費用     | 仕入高、旅費交通費、消耗品費 | 借方           |

## 貸借一致チェック

画面上部に貸借一致のチェック結果が表示されます。

- **緑色（貸借一致）**：借方合計と貸方合計が一致しています
- **赤色（貸借不一致）**：仕訳に誤りがある可能性があります

> **WARNING**: 貸借不一致の場合は、仕訳帳で各仕訳の借方・貸方が一致しているか確認してください。

## 小計と合計

各グループの小計と、全体の合計が表示されます。

- グループ行（薄いグレー背景）：グループごとの小計
- 合計行（青い背景）：すべての科目の合計

## CSVエクスポート

試算表をCSVファイルとしてダウンロードできます。

1. 年度と表示モードを選択
2. 「CSV」ボタンをクリック
3. ファイルが自動的にダウンロードされます

> **INFO**: ファイル名は「試算表\_[年度].csv」形式です。選択中の表示モードに応じた列が出力されます。

## 印刷・PDF保存

試算表を印刷またはPDFとして保存できます。

1. 年度と表示モードを選択
2. 「保存」または「印刷」ボタンをクリック
3. ブラウザの印刷ダイアログが表示されます

> **TIP**: 印刷ダイアログで「PDFとして保存」を選択すると、PDFファイルとして出力できます。確定申告の添付資料として活用できます。
