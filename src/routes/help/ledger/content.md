# 総勘定元帳

## 総勘定元帳とは

総勘定元帳は、勘定科目ごとに取引を時系列で一覧表示し、残高の推移を確認できる帳簿です。
仕訳帳に記録された取引を、科目別に整理して表示します。

> **INFO**: 確定申告や決算時に、各科目の取引履歴と残高を確認するために使用します。

## 基本的な使い方

1. サイドバーから「総勘定元帳」を選択
2. 画面上部のドロップダウンで年度を選択
3. 勘定科目を選択すると、その科目の取引一覧が表示されます

## 科目の選択

科目タイプ別にグループ化されたバッジから、表示したい科目をクリックして選択します。

- 資産、負債、純資産、収益、費用の順にグループ表示
- 選択中の科目は青色でハイライト表示

> **TIP**: 使用されている科目のみが表示されます。取引がない科目は表示されません。

## 表示項目

| 項目 | 説明 |
|------|------|
| 日付 | 取引日（月/日形式） |
| 摘要 | 取引の内容 |
| 相手科目 | 対となる勘定科目 |
| 借方 | 借方金額 |
| 貸方 | 貸方金額 |
| 残高 | 取引後の累計残高 |

> **INFO**: 複合仕訳の場合、相手科目は「諸口」と表示されます。

## 残高の計算

残高は勘定科目の種別によって計算方法が異なります。

| 種別 | 残高の増加 | 残高の減少 |
|------|------------|------------|
| 資産 | 借方 | 貸方 |
| 負債 | 貸方 | 借方 |
| 純資産 | 貸方 | 借方 |
| 収益 | 貸方 | 借方 |
| 費用 | 借方 | 貸方 |

## CSVエクスポート

表示中の科目データをCSVファイルとしてダウンロードできます。

1. エクスポートしたい科目を選択
2. 「CSV」ボタンをクリック
3. ファイルが自動的にダウンロードされます

> **INFO**: ファイル名は「総勘定元帳_[科目名]_[年度].csv」形式です。Excelや他の会計ソフトで開くことができます。

## 印刷・PDF保存

表示中の元帳を印刷またはPDFとして保存できます。

1. 印刷したい科目を選択
2. 「保存」または「印刷」ボタンをクリック
3. ブラウザの印刷ダイアログが表示されます

> **TIP**: 印刷ダイアログで「PDFとして保存」を選択すると、PDFファイルとして出力できます。確定申告の添付資料として活用できます。
