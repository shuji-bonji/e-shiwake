# WebMCP サンプルプロンプト集

> e-shiwake の WebMCP（Chrome 拡張 AI アシスタント）で使える指示の例

## 効果的なプロンプトのコツ

会話ログの分析から、以下の点を押さえると AI の無駄な確認が減りスムーズに操作できます。

1. **年度を明示する** — 「2月の仕訳」より「2026年2月の仕訳」
2. **勘定科目と支払方法を含める** — 「消耗品を追加」より「消耗品費 / 未払金で追加」
3. **1回で必要情報を全部渡す** — 日付・摘要・金額・取引先・支払方法をまとめて指示
4. **ブラウザUI操作の依頼は避ける** — 「検索バーに入力して」ではなく「検索して」

## 仕訳の作成

### 基本（経費・現金払い）

```
2026-02-23、消耗品費 3,980円、現金払い、取引先 Amazon、摘要「USBケーブル」で仕訳を作成して
```

### 基本（経費・クレジットカード＝未払金）

```
2026-02-20、旅費交通費 1,200円、未払金、取引先 JR東日本、摘要「電車代」で仕訳を追加して
```

### 家事按分（複合仕訳）

```
2026-02-14のNTTフレッツ光 6,985円を仕訳して。
通信費 2,305円（事業分）と事業主貸 4,680円（家事分）に按分、貸方は未払金で。
```

### 売上計上

```
2026-02-01、売上高 550,000円、売掛金として計上。
取引先は株式会社ヘルスベイシス、摘要「BIMツール開発支援 2月分」
```

### 入金処理

```
2026-03-05、普通預金に550,000円入金。
売掛金の回収、取引先は株式会社ヘルスベイシス、摘要「BIMツール開発支援：入金」
```

### 生活費の引き出し

```
今日の日付で、事業主貸 300,000円 / 普通預金 300,000円、摘要「生活費」で仕訳を追加して
```

### サブスク経費

```
2026-02-12、情報処理費 733円 / 未払金、取引先 DeepL、摘要「DeepL利用料」で仕訳して
```

## うまくいかないパターン（注意）

| やりたいこと     | NG プロンプト                                    | 理由                                    |
| ---------------- | ------------------------------------------------ | --------------------------------------- |
| ファイル読み込み | 「ダウンロードフォルダのデータをインポートして」 | AI はローカルファイルにアクセスできない |
| UI直接操作       | 「検索バーに2月と入力して」                      | AI はブラウザUIを操作できない           |
| 設定変更         | 「ダークモードにして」                           | WebMCPにUI操作ツールはない              |
| 記憶の保持       | 「この操作手順を覚えて」                         | AI にセッション跨ぎの記憶機能はない     |
| PDF紐付け        | 「証憑PDFを添付して」                            | WebMCPにファイル操作ツールはない        |
